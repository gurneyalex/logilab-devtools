.. contents::


Description
```````````
Cet utilitaire permet la création / maintenance automatique des fichiers de
configuration *debian*.

Voir les variables de la section "Debian" de la documentation_ du fichier
__pkginfo__.py.


Synopsis
````````

::

    USAGE:
	   debianize [OPTIONS] <python versions>
	      create / update debian files (debian mode). If no python versions are
	      provided, default to 2.3 2.4 (on the first run). Used options are
	      saved in a config file (debian/debianizerc) so you don't have to
	      provide them on futur runs.
	      The <python version> arguments is only used with the python-library
	      handler.  

	   debianize [OPTIONS] prepare 
	      create a directory to build the package (prepare mode).

    Both commands should be run in the root of the source package.

    OPTIONS:
      -h / --help
	   display this help message and exit.

      -o / --orig
	   create a .orig.tar.gz (only in prepare mode).



Exemple d'utilisation
`````````````````````

::

    $ cd logilab/pyreverse
    $ debianize --logilab --debian-name reverse 2.2 2.3
    $ cd `debianize prepare --orig`
    $ fakeroot make -f debian/rules binary


va mettre à jour les fichiers de configuration debian, créer un répertoire
correspondant aux besoins de la construction d'un paquet debian (ainsi que le
.orig.tar.gz), s'y placer et enfin lancer la construction du paquet.
