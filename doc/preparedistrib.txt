.. contents::


Description
```````````
Cet utilitaire prépare la création d'une distribution. Il effectue les actions
suivantes :

1. éxécution de pylint_.

2. éxécution de checkpackage_.

3. lancement des tests du paquet.

4. fermeture du ChangeLog s'il existe (voir l'utilitaire changelog_).

5. construction de la documentation.

6. génération de l'annonce si le fichier announce.txt existe.

Toutes ces étapes sont effectuées de manière interactive (i.e. le script demande
s'il doit ou non effectuer chaque étape).  


Synopsis
````````

::

    USAGE: preparedistrib <source_directory>


Exemple d'utilisation
`````````````````````

::

    preparedistrib logilab/pyreverse

va créer lancer la préparation d'une distribution du module *pyreverse*.
