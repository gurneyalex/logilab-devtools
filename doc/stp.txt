Description
```````````
Un très simple préprocesseur de texte. Utilisé par exemple dans *preparedistrib* 
pour générer l'annonce.


Synopsis
````````

::

    USAGE: stp input-file PATTERN VALUE [PATTERN VALUE]...



Exemple d'utilisation
`````````````````````

::

    stp announce.txt WEB http://www.perdu.com MAILINGLIST labas@perdu.com

va imprimer sur la sortie standard le contenu du fichier *announce.txt* ou
toutes les occurences de %WEB% auront été remplacées par *http://www.perdu.com
et les occurences de %MAILINGLIST% par *labas@perdu.com*.
