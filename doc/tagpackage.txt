=========================

.. contents::


Description
```````````
Cet utilitaire permet de gérer les tags de distributions de manière standard. Il
va générer le tag correspondant à la version courante et l'appliquer si vous le
confirmer. Il gère des tags pour les versions sources et les versions debian.


Synopsis
````````

::

    USAGE: tagpackage <package directory>


Exemple d'utilisation
`````````````````````

::

    tagpackage logilab/pyreverse


va vous demander si vous désirer tagger le paquet pyreverse pour la version
source et la version debian courante.
