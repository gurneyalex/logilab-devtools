Description
```````````
Cet utilitaire aide à la création d'une distribution. Il effectue les actions
suivantes :

1. vérification que le répertoire de travail est synchroniser avec le repository.

2. vérification que aucun fichier n'est en édition.

3. construction du paquet à l'aide de buildpackage_.

4. ajout des tags nécessaires dans le CVS.

Toutes ces étapes (sauf la 3.) sont effectuées de manière interactive (i.e. le
script demande s'il doit ou non effectuer chaque étape). 


Synopsis
````````

::

    USAGE: makedistrib <source_directory> <bdist_wininst|sdist|deb>


Exemple d'utilisation
`````````````````````

::

    makedistrib logilab/pyreverse deb

va créer lancer la construction d'un paquet debian pour *pyreverse*.
