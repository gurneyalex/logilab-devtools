L'utilitaire *buildpackage*
===========================

:Author: Sylvain Thénault
:Organization: Logilab
:Version: $Revision: 1.1 $
:Date: $Date: 2003-09-19 11:50:08 $

.. contents::


Description
```````````
Cet utilitaire aide construit une distribution pour un paquet (source,
debian). 


Synopsis
````````

::

    USAGE: buildpackage <source_directory> <bdist_wininst|sdist|deb>


Exemples d'utilisation
``````````````````````

::

    buildpackage logilab/pyreverse deb

va créer lancer la construction d'un paquet debian pour *pyreverse* et
placer les fichiers générés dans le sous-répertoire *dist* du
répertoire courant.

::

    buildpackage logilab/pyreverse sdist

ou simplement

::

    buildpackage logilab/pyreverse 

va créer lancer la construction d'un paquet source pour *pyreverse* et
placer les fichiers générés dans le sous-répertoire *dist* du
répertoire courant.
